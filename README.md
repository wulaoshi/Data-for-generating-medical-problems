# Data-for-generating-medical-problems
医学询问问题生成模型训练数据。
## 介绍
- 本项目提供了一个用于训练医学询问模型的数据，共106万条。
- 该数据是已经经过清洗、处理和构造，能直接用于训练GPT2，Seq2Seq等模型。
- 上述模型详见论文 [未写](www.baidu.com)。
